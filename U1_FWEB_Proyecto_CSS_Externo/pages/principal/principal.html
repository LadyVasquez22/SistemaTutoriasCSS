Bienvenido al sistema

Este sitio web permite a los estudiantes solicitar tutorias academicas de manera organizada, seleccionando docente, fecha, hora, modalidad y motivo de la cita.

La propuesta busca mejorar la comunicacion entre estudiantes y docentes, evitando confusiones en horarios y solicitudes informales.

Servicios principales

  • Registro de usuarios Ingreso ordenado de estudiantes al sistema.
  • Solicitud de tutorias Formulario para seleccionar docente, fecha, hora y modalidad.
  • Consulta de solicitudes Tabla para revisar tutorias registradas y su estado.
  • Contacto academico Canal para dudas, sugerencias o problemas tecnicos.

Recursos academicos

Horarios

Organizacion de espacios disponibles para atencion estudiantil.

Docentes

Informacion basica para elegir el tutor adecuado segun la materia.

Modalidades

Atencion presencial o virtual segun la necesidad del estudiante.

Seguimiento

Revision del avance de solicitudes pendientes, aceptadas y finalizadas.

Comunicacion

Formulario para enviar dudas al area academica.

Soporte

Orientacion para completar correctamente cada cita.

Informacion importante

Las tutorias deben solicitarse con anticipacion para que el docente pueda revisar su disponibilidad y confirmar la atencion.

Aviso

El horario de atención es de lunes a viernes de 08:00 a 18:00.
